Acerca de esta aplicación

¿Qué es PelículasApp?

PelículasApp es una aplicación web sencilla y práctica que permite buscar películas y series utilizando la API de OMDb. Puedes acceder a detalles completos de cada título, como sinopsis, actores, directores, géneros y más.

Tecnologías utilizadas

Esta aplicación está construida con Angular, un potente framework de desarrollo frontend, y con Bootstrap para un diseño atractivo y responsivo. Utilizamos la API OMDb para acceder a una vasta base de datos de películas y series.
